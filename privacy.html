2. Tipologia di dati trattati

HACCP Pro è un'applicazione web (PWA) che funziona interamente offline. Non raccogliamo, memorizziamo o trasmettiamo alcun dato personale degli utenti sui nostri server.

Tutti i dati inseriti nell'applicazione (dati aziendali, registrazioni temperature, tracciabilità lotti, pulizie) vengono salvati esclusivamente in locale sul dispositivo dell'utente tramite la tecnologia IndexedDB del browser.

3. Dati NON raccolti

HACCP Pro NON raccoglie:

  • Dati di navigazione o indirizzi IP
  • Dati di geolocalizzazione
  • Informazioni sul dispositivo
  • Dati personali di alcun tipo
  • Dati analitici o statistiche d'uso

4. Archiviazione locale dei dati

I dati inseriti nell'applicazione HACCP Pro vengono memorizzati esclusivamente sul dispositivo dell'utente utilizzando:

  • IndexedDB: database locale del browser per i dati operativi (temperature, lotti, pulizie, dati aziendali)
  • localStorage: per preferenze minime dell'interfaccia (es. consenso cookie)
  • Cache API (Service Worker): per il funzionamento offline dell'applicazione

Nessuno di questi dati viene trasmesso a server esterni. L'utente può cancellare tutti i dati in qualsiasi momento attraverso le impostazioni del browser (cancellazione dati siti web).

5. Cookie

HACCP Pro utilizza esclusivamente cookie tecnici strettamente necessari al funzionamento dell'applicazione. Non vengono utilizzati cookie di profilazione, analitici o di terze parti. Per maggiori dettagli, consulta la nostra Cookie Policy.

6. Trasferimento dati a terze parti

HACCP Pro non trasferisce alcun dato a terze parti. L'unica interazione esterna avviene al momento dell'acquisto della licenza, gestita tramite la piattaforma PayPal, soggetta alla Privacy Policy di PayPal.

7. Diritti dell'utente (GDPR)

Ai sensi del Regolamento UE 2016/679 (GDPR) e del D.Lgs. 196/2003 (Codice Privacy italiano), l'utente ha diritto di:

  • Accedere ai propri dati personali
  • Rettificare dati inesatti
  • Cancellare i propri dati (“diritto all'oblio”)
  • Limitare il trattamento
  • Opporsi al trattamento
  • Portabilità dei dati

Poiché HACCP Pro non raccoglie né memorizza dati personali su server remoti, tutti i dati sono già sotto il pieno controllo dell'utente sul proprio dispositivo. L'utente può visualizzare, modificare o eliminare i propri dati in qualsiasi momento direttamente dall'applicazione o dalle impostazioni del browser.

8. Sicurezza

La sicurezza dei dati è garantita dall'architettura stessa dell'applicazione: non esistendo trasmissione di dati verso server esterni, non sussistono rischi legati a violazioni di database remoti o intercettazioni di rete. I dati restano esclusivamente sul dispositivo dell'utente, protetti dalle misure di sicurezza del browser e del sistema operativo.
